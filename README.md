# DOIN - Dados de Ocupantes Informado na Nuvem

Projeto desenvolvido para monitoramento inteligente da ocupação do transporte público urbano.

O sistema coleta e processa dados de entrada e saída de passageiros, permitindo análises sobre:

* Taxa de ocupação
* Horários de pico
* Origem e destino de passageiros
* Superlotação
* Linhas com baixa demanda
* Relatórios operacionais

## Objetivo

Auxiliar gestores públicos e empresas de transporte na tomada de decisões baseadas em dados, promovendo maior eficiência e qualidade no transporte coletivo.

## Tecnologias

* IoT
* Sensores Infravermelhos
* Arduino / Microcontroladores
* Automação
* APIs
* Banco de Dados
* Análise de Dados
